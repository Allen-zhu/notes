## 结论
保留最常用生命周期命令，避免临时查找。

## 操作
- `docker ps`：查看正在运行的容器；常用 `docker ps --format "table {{.Names}}\t{{.Status}}"` 快速看名称和状态。
- `docker images`：查看本地镜像列表；可配合 `docker images | rg "<image>"` 定位目标镜像。
- `docker compose up -d`：后台启动 `compose` 编排的服务；首次启动前确认当前目录和 `compose` 文件环境变量是否正确。

## 注意事项
删除镜像、容器、卷等破坏性命令需单独分节并标风险。
